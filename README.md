# PHP-GTK 2024 Example
*Using PHP 8.2.4*

This project exposes GTK to PHP. Run the demo with the following command on your terminal:

```bash
git clone GIT_URL
cd php-gtk/
node .
```

## Updating php-wasm

Clone the project & enter the directory:
```bash
git clone GIT_URL
cd php-gtk/
```
## Updating php-wasm

```bash
npm run update-php
```
